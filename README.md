# Credit Scoring Model - CodeAlpha

Projet de stage Machine Learning : Prediction de la solvabilite bancaire.

## Description

Modele de scoring credit avec 3 algorithmes :
- Regression Logistique
- Arbre de Decision
- Random Forest

## Dataset

German Credit Dataset (UCI) - 1000 instances, 20 features
