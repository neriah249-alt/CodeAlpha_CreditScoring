# 🏦 Credit Scoring Model - CodeAlpha

Projet de stage Machine Learning : Prédiction de la solvabilité bancaire.

## 📋 Description

Ce projet implémente un **modèle de scoring crédit** qui prédit la solvabilité d'un individu à partir de ses données financières. Trois algorithmes de classification sont comparés :

- **Régression Logistique**
- **Arbre de Décision**
- **Random Forest**

## 📊 Dataset

**German Credit Dataset** (UCI Machine Learning Repository)
- 1000 instances
- 20 features
- Cible binaire : Bon crédit (0) / Mauvais crédit (1)
